# Java-Practice
These are some practice problems that I worked on in my class. These problems are not worthy of a repo for their own, So I figured a repo that collects all the problems I worked on in free time and ones I worked on in class would be be fitting. 
## Topics Covered
- Control flow (loops, conditionals)
- String manipulation
- Arrays and sorting
- Object-oriented programming

## Problems
| Problem | Concepts | 
|---------|----------|
| FizzBuzz | Loops, modular arithmetic |
| Palindrome Checker | Strings, conditionals |
| Bubble Sort | Arrays, nested loops |
|Caculator| Loops, if else |
|Target| Loops, boolean, varibles| 

## How to Run
1. Clone the repo: `git clone <your-repo-url>`
2. Open in your IDE (IntelliJ, Eclipse, VS Code)
3. Navigate to the problem file and run `main()`

## Tools
- Java 17
- IntelliJ IDEA
